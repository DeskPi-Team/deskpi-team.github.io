Getting Start

.

For order please visit: DeskPi.com.

How to Install Deskpi driver

    1. Flash the latest version of supported OS to your TF(microSD) card with etcher flash imaging tool
    1. Insert TF(MicroSD) card to DeskPi card slot and connect the power supply(which come with the package.)
    1. Connect DeskPi to internet. (You need to configure WLAN country in order to enable Wi-Fi adapter)
    1. Open a terminal by press icon on the task bar or just press "Ctrl+alt+T.
    1. Typing those command to install the driver automatically.(Raspberry Pi will reboot after installing.)

For Raspbian and RetroPie OS.

cd ~
git clone https://github.com/DeskPi-Team/deskpi.git
cd ~/deskpi/
chmod +x install.sh
sudo ./install.sh

For Ubuntu-mate OS

cd ~
git clone https://github.com/DeskPi-Team/deskpi.git
